<h1>Genesis Core</h1>
Um emulador web exclusivamente para sega genesis (mega drive), onde você pode pegar a _sua_ rom e ler no site para você jogar direto no navegador
<p></p>
<P>Observações:</p> 
<P>não tem como salvar os jogos,</p> 
<p>use o save state como um arquivo local que o site consegue criar,</p> 
<p>nenhuma rom vai estar ou vai ser disponibilizada no site,</p> 
<p>jogos de sega CD e 32X não funcionam</P>
<p>Somente da para jogar com teclado, caso queira jogar no mobile, vai ter que usar teclado</p>
<p>e sim é possível zerar jogos nisso (porque uma vez zerei um jogo sem querer)</p>
